#  Webová prezentace: Topenář Jan Hruška

Jednoduchý a moderní jednostránkový web (One-page website) vytvořený pro instalatérské a topenářské služby. Projekt byl zaměřen na čistý design, rychlost načítání a plnou responzivitu pro mobilní zařízení.

##  Živá ukázka
Na web se můžete podívat zde: [https://janhruska-git.github.io/topenarstvi-hruska/](https://janhruska-git.github.io/topenarstvi-hruska/)

##  Použité technologie
* **HTML5** - Struktura obsahu a sémantika.
* **CSS3** - Stylování, Flexbox pro rozvržení a Media Queries pro mobilní verzi.
* **Google Fonts** - Použití fontů *Oswald* a *Roboto* pro moderní vzhled.
* **GitHub Pages** - Hosting a nasazení webu do ostrého provozu.

##  Klíčové funkce
* **Responzivní design:** Web se automaticky přizpůsobí mobilům, tabletům i počítačům.
* **Sticky navigace:** Menu zůstává nahoře i při skrolování pro snadnou orientaci.
* **Smooth Scroll:** Plynulé přejíždění mezi sekcemi (Domů, Nabídka, Kontakt).
* **Mobilní tlačítko:** Speciální fixní tlačítko pro rychlé zavolání přímo z telefonu.

## Náhledy
*Web obsahuje sekce pro provozní dobu, přehled nabízených služeb (tepelná čerpadla, kotle, rekuperace) a galerii realizovaných prací.*
